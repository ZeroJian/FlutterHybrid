

### FlutterHybrid

[开发指南](https://github.com/ZeroJian/FlutterHybrid/tree/master/开发指南)

产物集成方式集成 Flutter 项目

1.不依赖原生项目,对原生项目无侵入

2.不需要 Flutter 开发环境

3.包含了 Flutter 插件原生代码的集成

### 包含文件

1.MyFlutterSDK

Flutter build ios 生成的产物, 包含 APP.framework Flutter.framework

2.MyApp

原生项目 Demo

3.FlutterPluginRegistrant, flutter_boost, xservice_kit

Flutter 插件原生代码 pod

4.my_flutter

Flutter 开发项目
